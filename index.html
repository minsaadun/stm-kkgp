Kenapa pilih program ini?

Program ini sesuai untuk pelajar yang berminat membina asas kemahiran IT sebelum bekerja atau menyambung pengajian ke tahap lebih tinggi.

💻

Asas IT yang luas

Pelajar didedahkan kepada hardware, software, database, programming, operating system, networking dan cybersecurity.

🛠️

Fokus kemahiran praktikal

Pembelajaran merangkumi teori dan amali supaya pelajar boleh melaksanakan tugasan IT berkaitan alam pekerjaan.

🏢

Latihan Industri

Pelajar menjalani latihan industri untuk mendapatkan pengalaman sebenar dalam persekitaran kerja bidang IT.

Matlamat Program

Melahirkan sokongan teknikal IT yang berpengetahuan, berkemahiran digital, bertanggungjawab dan mampu menyesuaikan diri dengan perkembangan teknologi semasa.

✓ Kemahiran teknikal komputer dan rangkaian
✓ Komunikasi dan kerja berpasukan
✓ Etika profesional dan keusahawanan

Sesuai untuk siapa?

Pelajar lepasan SPM yang suka mencuba teknologi, membaiki komputer, membina laman web, memahami rangkaian, belajar coding dan meneroka keselamatan siber.

Laluan Pembelajaran

Gambaran ringkas perjalanan pelajar sepanjang program pengajian

1

Semester 1

Basic Mathematics, Database Fundamentals, Programming Fundamentals, Computer Architecture, Systems Analysis and Design Fundamentals.

2

Semester 2

Operating Systems, Network and Data Communications, Web Development, Object-Oriented Programming, Network Technology.

3

Semester 3

Cybersecurity Fundamentals, Mobile Application Development, Introduction to Internet of Things (IoT), Essential Project Management, Network Administration, Computer Repair and Maintenance

4

Semester 4

Latihan Industri untuk pengalaman kerja sebenar.
